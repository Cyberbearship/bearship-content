# 製作流程

## 拍攝流程

1. **路線規劃**：確認起點、終點、拍攝時段
2. **設備檢查**：攝影機電量、固定支架、儲存空間
3. **拍攝執行**：記錄 FSD 版本、天氣、時間
4. **備份原始檔**：拍攝後立即備份

## 後製流程

1. **挑選片段**：從原始檔中挑選畫面乾淨的段落
2. **快轉壓縮**：將 20-40 分鐘路線壓縮至 5-10 分鐘
3. **調色**：提升對比、優化夜間畫質
4. **添加疊加資訊**：右下角顯示車型 + FSD 版本
5. **配樂**：選擇賽博龐克風格音樂
6. **輸出**：4K、16:9、24fps 或 30fps

## 發布流程

1. **標題撰寫**：清楚描述 FSD 版本、天氣、路線
2. **描述撰寫**：加入路線資訊、FSD 版本、相關連結
3. **縮圖設計**：吸引點擊的視覺設計
4. **上傳發布**：無固定排程，興趣驅動
