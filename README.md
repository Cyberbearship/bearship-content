# Bearship

> Tesla FSD + SpaceX 沉浸式第一人稱駕駛紀錄片頻道

---

## 專案目標

建立一個**可持續、可擴充**的內容創作專案，記錄台灣 Tesla FSD 的真實駕駛體驗，並延伸至 SpaceX / Tesla 相關科技內容。

### 核心原則

- **品質優先**：興趣驅動，不強求固定排程
- **沉浸式體驗**：極簡視覺 + 賽博龐克音樂
- **全球視野**：避開老舊建築，呈現現代化台灣城市景觀
- **長期可維護**：模組化文件結構，方便未來擴充

## 頻道定位摘要

| 項目 | 內容 |
|------|------|
| **頻道名稱** | Bearship |
| **目標觀眾** | 全球 Tesla 車主 / FSD 潛在買家（英文為主） |
| **視覺風格** | 極簡沉浸式，只在右下角顯示車型 + FSD 版本 |
| **聽覺風格** | 純音樂沉浸 + 保留部分真實環境音（無旁白） |
| **音樂風格** | 賽博龐克 / 電子音樂 |
| **影片長度** | 5-10 分鐘（快轉壓縮） |
| **上傳頻率** | 興趣驅動，無固定排程 |
| **差異化** | 台灣現代化路線 + 賽博龐克音樂 |

### 路線規劃

**路線 A：高速公路夜間路線**
- 台中市北屯區 → 國道1號 → 竹科/竹南科學園區
- 拍攝時段：夜間（8pm 之後）
- 特色：速度感、賽博龐克氛圍

**路線 B：重劃區白天路線**
- 單元十二 → 機捷特區 → 十四期
- 拍攝時段：白天黃金時段
- 特色：現代建築、光影效果

### 參考頻道

- **Whole Mars Catalog**：純行車畫面 + 沉浸感、標題風格
- **AIDRIVR**：第一人稱純駕駛體驗
