# 設備與拍攝

## 攝影機選擇

**需求**：
- 4K 畫質
- 高動態範圍（HDR）
- 良好夜視能力
- 適合車內架設（小巧、穩定）

**推薦類型**：
- 運動攝影機（GoPro、Insta360 等）
- 微單 / 無反相機（Sony、Fujifilm 等）
- 專用行車記錄器（支援外部麥克風）

## 架設位置

**駕駛與副駕駛之間**：
- 視角接近駕駛視角，構圖平衡
- 完整呈現中控螢幕 FSD 介面
- 需控制角度，避免駕駛入鏡

**注意事項**：
- 確保不影響駕駛安全
- 避免遮擋後視鏡視線
- 固定穩固，避免行駛中晃動

## 畫面設定

- **畫面比例**：16:9（YouTube 標準）
- **解析度**：4K（3840×2160）
- **幀率**：24fps 或 30fps（電影感 vs 流暢感）
- **畫面右下角疊加**：車型 + FSD 版本（後製或即時）
